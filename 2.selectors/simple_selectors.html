CSS 선택자(CSS 선택자)

1.전체 선택자(*,Universal Selector) -전체 폰트 적용

2.타입 선택자(Type Selector), 해당 타입(태그명,태그의 이름)은 전부 적용됨

그냥 p tag

3.클래스 선택자(Class Selector),attribute로 class로 씀

Class Selector 텍스트

Class Selector 텍스트는 중복 적용됨

4. 아이디 선택자(Id Selector), 기호는 #(hashtag) attribute로 id로 씀

Id 선택자도 중복 적용은 되지만(문법적으로), 한번만 사용을 권장
